# Albanian Language Technology Development Plan (2026–2028)

**Version 0.2 | June 2026**
**Cross-Border Initiative: Albania • Kosovo • North Macedonia**

---

Albanian is spoken by approximately 8 million people but remains severely underrepresented in natural language processing and AI. This plan is a 3-year accelerated roadmap to build foundational language technology infrastructure for Albanian — corpora, NLP tools, speech recognition, language models, and public service integrations — through coordinated cross-border effort.

## What this plan covers

- **Language resources:** A 60-million-token text corpus, 1,000 hours of annotated speech (targeting 1,000+ hours via Mozilla Common Voice), and 4 million parallel translation pairs
- **NLP tools and models:** 6–8 open-source tools including tokenizers, POS taggers, NER systems, machine translation, ASR, and an Albanian LLM via continual pretraining
- **Public service integration:** Language technologies deployed in 8–10 e-government platforms in Albania and Kosovo
- **Research capacity:** 12 PhD scholarships, 6 postdoctoral fellowships, and microgrants for 30+ individual researchers
- **Evaluation infrastructure:** Open Albanian LLM Leaderboard with translated standard benchmarks (MMLU, HellaSwag, ARC)

**Budget:** €6.5–8.2 million over 3 years (Albania and Kosovo governments + EU co-funding via IPA III, Horizon Europe, Digital Europe, CEF)
